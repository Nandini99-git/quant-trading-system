# AI-Powered Quant Trading System

An end-to-end quantitative trading research project built as part of an AI-Powered Quant Trading Systems Internship.

## Project Overview

This project aims to build a research-grade quantitative trading system capable of:

* Data Engineering Pipeline
* ATR-Based Adaptive Renko Strategy
* Machine Learning Trade Validation
* Risk Management Engine
* Historical Backtesting
* Forward Testing
* Paper Trading Simulation
* AI-Based Strategy Analysis

The objective is to design a reproducible, modular, and production-ready trading research framework rather than a simple trading bot.
